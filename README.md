## Getting started

Install Deno: [installation guide](https://deno.land/manual/getting_started/installation)  

Run the game server  
`deno run --allow-net=:3000 --allow-read main.ts --port=3000`

Use URL to connect: `http://localhost:3000`

Enjoy
