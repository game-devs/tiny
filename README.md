## Getting started

Install Deno  
Powershell:  
`iwr https://deno.land/x/install/install.ps1 -useb | iex`  
Other:  
[deno guide](https://deno.land/#installation)

Run the game server  
`deno run --allow-net=:3000 --allow-read main.ts --port=3000`

Use URL to connect: `http://localhost:3000`

Enjoy
